Step 1: Setting up Git

Here's an example of how to start your HTML file:

<!DOCTYPE yaml>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my very first paragraph.</p>
</body>
</html>

Try copying this code into a new file named `index.html` (or `exercise1.html`) and opening it in your browser.

Step 2: Add More Elements

Now, try adding an image and a list to your page:

<!-- Inside your <body> tag -->
<img src="https://placehold.co/200x100/A0D911/000000?text=Placeholder" alt="A simple placeholder image" class="rounded-md"/>
<h2>My Favorite Fruits</h2>
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>

Experiment with different tags like `<a>` for links, `<div>` for containers, and `<strong>` for bold text.
